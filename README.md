# vue-admin

Vue2.0 企业级管理后台项目快速开发原型，简单、实用、克制。

### 如何启动

- yarn
- yarn serve
- yarn build:client
- yarn build:server
- yarn build:win 

### 相关技术

- vue
- vue-router
- vuex
- vue-server-renderer
- sass
- fontawesome
- element-ui
- axios
- jsonp
- echarts
- wangeditor
- jsonp

# vue-admin

Vue2.0 企业级管理后台项目快速开发原型，简单、实用、克制。

### 如何启动

- yarn 安装依赖包
- yarn serve 开发调试
- yarn build:win SSR构建
- yarn ssr 运行SSR
- yarn lint 代码风格检测&自动修复

### 相关技术

- vue
- vue-router
- vuex
- vue-server-renderer
- sass
- fontawesome
- element-ui
- axios
- jsonp
- echarts
- wangeditor
- jsonp

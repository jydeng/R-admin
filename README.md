## vue-admin

Vue2.0 企业级管理系统快速开发原型

## 功能

* Vue-CLI 3  
* Vue-Router
* Vuex
* 基于token认证的权限系统
* axios、jsonp封装
* cookie、sessionStorage封装
* FontAwesome
* 富文本编辑器
* 自定义主题
* 服务端渲染
* Echarts图表


## 使用

- 初始化
```cmd
git clone  git@github.com:jydeng/vue-admin.git
cd vue-admin
yarn
```

- 开发调试
```cmd
yarn serve 
```

- 构建
```cmd
yarn lint
yarn build
```

- 服务端渲染构建&启动Koa服务器
```cmd
yarn build:win 
node server
```
